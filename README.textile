h1. Textile Buttons

"github.com/cementhorses/textile_buttons":http://github.com/cementhorses/textile_buttons

A cross-browser utility for working with textile. Easily create a toolbar to facilitate textile (or markdown) entry.

<hr />

h2. Features

* Works in IE7, Firefox 3, Safari 3
* Will apply markup to highlighted text, or insert example text if no selection
